# MolCap text target, seed 7777

## Decision

Do not submit this recipe. The run completed successfully and preserved the
frontier's tile-discrimination metrics, but its locked final score was
`0.6652239193862792`, below the `bsc-s7777-k10` reference score of
`0.6659107210081501` by `0.0006868016218708`.

The result supports the patch-routed, late-ramped decoupling mechanism, but it
rejects the current per-patient MiniLM caption target as a frontier-beating
objective at weight `0.03`.

## Interpretation

The intended isolation worked: linear was effectively unchanged and kNN
improved, while slide AUC increased. The failure was target-specific rather
than an optimization failure: the molecular AUC and survival categories moved
down, and segmentation also lost enough to erase the slide gain.

The next justified MolCap experiment is patient-centroid supervision aligned
to the exact mean-pooled slide representation consumed by slide probes. It
should be tested against this run at the same seed before any multi-seed
campaign. Increasing the current caption weight is not justified by these
results.
